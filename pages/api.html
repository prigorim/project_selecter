Метод basket

Метод работы с корзиной.
Добавление, удаление, изменение товарной позиции в корзине пользователя.
Корзина пользователя не существует автономно, она связана в обе стороны с корзиной на сайте.
get
Получает корзину текущего пользователя.
Путь: /api/v1/basket/get/
PRICE_CODE - тип цены товаров в корзине
ID - id товара в корзине
NAME - название товара
PRODUCT_ID - id товара
QUANTITY - кол-во
DELAY - отложен или нет
CAN_BUY - возможна покупка
CURRENCY - валюта
PRICES: {
BASE_PRICE - цена без скидки
PRICE - цена со скидкой
DISCOUNT - сумма скидки
},
PROPERTY: [] - набор свойств
add
Добавляет товар в корзину для текущего пользователя.
Путь: /api/v1/basket/add/
id_item - id товара
quantity - кол-во (добавляет)
PRICE_CODE - тип цены товара
BASKET_PRICE_SUM - сумма товаров в корзине с учетом скидок
BASKET_BASE_PRICE_SUM - сумма товаров в корзине без учета скидок
BASKET_COUNT - кол-во позиций в корзине
del
Удаляет товар из корзины для текущего пользователя.
Путь: /api/v1/basket/del/
id_item - id товара
PRICE_CODE - тип цены товара
ID - id товара в корзине
NAME - название товара
PRODUCT_ID - id товара
QUANTITY - кол-во
DELAY - отложен или нет
CAN_BUY - возможна покупка
CURRENCY - валюта
